Citation: citation field

Supports inline encoding

Citation of a resource

The citation field may have flags:

  • id flag (optional)
  • href flag (optional)
Properties
NameMetaschema typeCardinalityDescription / Remarks
idflag (zero or one)
hrefflag (zero or one)

Semantics of any contents within citation are implicit, not encoded. To mitigate problems establishin matches with bibliograpical databases, it is recommended a canonical identifier such as a DOI be provided for any citation.

  • {% highlight xml %} Here is a work cited Some citation of some sort {% endhighlight %}
  • {% highlight json %} { "citations" : [ { "RICHTEXT" : "Here is a work cited" }, { "RICHTEXT" : "Some **citation** of some sort" } ] } {% endhighlight %}

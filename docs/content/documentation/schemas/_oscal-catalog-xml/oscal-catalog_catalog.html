Catalog: catalog element

catalog is the root (containing) element of this schema.

A collection of controls

The catalog element has the following contents (in order):

  • id attribute (required)
  • model-version attribute (required)
  • title element (one)
  • metadata element (zero or one)
  • declarations element (zero or one)
  • references element (zero or one)
  • section element (zero or more)
  • A choice between
    • group element (zero or more)
    • control element (zero or more)

Catalogs may use section to subdivide the textual contents of a catalog.

  • {% highlight xml %} <catalog id="simple-example" model-version="0.99"> <title>A Miniature Catalog</title> <control id="single"> <title>A Single Control</title></control></catalog> {% endhighlight %}

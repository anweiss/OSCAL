Publication metadata: metadata element

metadata is the root (containing) element of this schema.

Information describing the publication or availability of this document

The metadata element has the following contents (in order):

  • title element (one)
  • author element (zero or more)
  • publication-date element (zero or one)
  • version element (zero or one)
  • doc-id element (zero or more)
  • prop element (zero or more)
  • hlink element (zero or more)
  • resource element (zero or more)
  • role element (zero or more)
  • party element (zero or more)
  • notes element (zero or one)
  • extra-meta element (zero or one)

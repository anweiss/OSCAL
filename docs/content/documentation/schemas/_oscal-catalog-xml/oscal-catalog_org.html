Organization: org element

An organization or legal entity (not a person), with contact information

The org element has the following contents (in order):

  • org-name element (zero or one)
  • short-name element (zero or one)
  • org-id element (zero or more)
  • address element (zero or more)
  • email element (zero or more)
  • phone element (zero or more)
  • url element (zero or more)
  • notes element (zero or one)

This element appears inside: party.

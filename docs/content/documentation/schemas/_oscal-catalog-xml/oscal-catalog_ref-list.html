Reference: ref-list element

A list of references to external resources. This object may be nested for reference lists composed of reference lists.

The ref-list element has the following contents (in order):

  • id attribute (optional)
  • title element (zero or one)
  • Prose contents (paragraphs and lists)
  • link element (zero or more)
  • ref element (zero or more)
  • ref-list element (zero or more)

This element appears inside: ref-list, back, section, group, control, subcontrol.

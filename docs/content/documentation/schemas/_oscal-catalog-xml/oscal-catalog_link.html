Link: link element

Supports inline encoding

A line or paragraph with a hypertext link

The link element may have attributes

  • href attribute (optional)
  • rel attribute (optional)

Works like an HTML anchor (a) except this is a line-oriented (block) element.

This element appears inside: ref-list, control, subcontrol, param, part.

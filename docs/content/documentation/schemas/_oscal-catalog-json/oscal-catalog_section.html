Section: section object

Allows the inclusion of prose content within a Catalog.

The section object has the following members (properties):

  • id property (optional)
  • class property (optional)
  • title object (one)
  • Prose contents (paragraphs and lists)
  • section object (zero or more)
  • references object (zero or one)

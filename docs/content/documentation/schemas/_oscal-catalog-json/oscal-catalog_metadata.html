Publication metadata: metadata object

metadata is the root (containing) object of this schema.

Information describing the publication or availability of this document

The metadata object has the following members (properties):

  • title object (one)
  • author object (zero or more)
  • publication-date object (zero or one)
  • version object (zero or one)
  • doc-id object (zero or more)
  • prop object (zero or more)
  • hlink object (zero or more)
  • resource object (zero or more)
  • role object (zero or more)
  • party object (zero or more)
  • notes object (zero or one)
  • extra-meta object (zero or one)

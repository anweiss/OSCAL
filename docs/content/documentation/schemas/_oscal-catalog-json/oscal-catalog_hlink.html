Hashed link: hlink object

A pointer to an external document with optional hash for verification

The hlink object has the following members (properties):

  • href property (required)
  • rel property (optional)
  • media-type property (optional)
  • title object (zero or one)
  • hash object (zero or more)

This is different from unstructured link, which makes no provision for a hash or formal title. It is also different from the element of the same name once proposed for HTML.

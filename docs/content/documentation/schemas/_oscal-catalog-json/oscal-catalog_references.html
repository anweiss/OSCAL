References: references object

A group of reference descriptions

The references object has the following members (properties):

  • id property (optional)
  • An array labelled links, containing link objects (optional)
  • An array labelled refs, containing ref objects (optional)

This object appears as a property on: catalog, section, group, control, subcontrol.

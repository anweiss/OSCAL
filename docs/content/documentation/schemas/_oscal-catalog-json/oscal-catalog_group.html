Control Group: group object

This object appears unlabelled in any array called groups

A group of controls, or of groups of controls.

The group object has the following members (properties):

  • id property (optional)
  • class property (optional)
  • title object (required)
  • An array labelled , containing param objects (optional)
  • An array labelled , containing prop objects (optional)
  • An array labelled , containing part objects (optional)
  • A choice between
    • An array labelled , containing group objects (optional)
    • An array labelled , containing control objects (optional)
  • ref-list object (optional)

Catalogs can use groups to provide collections of related controls or Control Groups. A group may have its own properties, statements, parameters, and references, which are inherited by all members of that group. Unlike a section, a group may not contain arbitrary prose directly, only inside its part or control components.

  • {% highlight json %} { "id" : "xyz", "title" : "My Group", "properties" : [ { "class" : "required", "STRVALUE" : "some property" } ], "controls" : [ { "id" : "xyz1", "title" : "Control" } ] } {% endhighlight %}

This object appears as a property on: catalog, group.

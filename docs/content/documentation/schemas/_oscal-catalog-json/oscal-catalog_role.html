Role: role object

Defining a role to be assigned to a party or agent

The role object has the following members (properties):

  • id property (required)
  • title object (zero or one)
  • short-name object (zero or one)
  • desc object (zero or one)

Permissible values to be determined closer to the application (e.g. by a receiving authority).

Catalog: catalog object

catalog is the root (containing) object of this schema.

A collection of controls

The catalog object has the following members (properties):

  • id property (required)
  • model-version property (required)
  • title object (one)
  • metadata object (zero or one)
  • declarations object (zero or one)
  • references object (zero or one)
  • section object (zero or more)
  • A choice between
    • group object (zero or more)
    • control object (zero or more)

Catalogs may use section to subdivide the textual contents of a catalog.

  • {% highlight json %} { "id" : "simple-example", "model-version" : "0.99", "title" : "A Miniature Catalog", "controls" : [ { "id" : "single", "title" : "A Single Control" } ] } {% endhighlight %}

A small catalog with a single control

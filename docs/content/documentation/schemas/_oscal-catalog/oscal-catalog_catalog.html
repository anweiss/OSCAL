Catalog: catalog assembly

catalog is the root (containing) element of this schema.

A collection of controls

The catalog assembly has the following contents (in order):

  • id flag (required)
  • model-version flag (required)
  • title field (one)
  • metadata assembly (zero or one)
  • declarations field (zero or one)
  • references assembly (zero or one)
  • section assemblies (zero or more)
  • A choice between
    • group assemblies (zero or more)
    • control assemblies (zero or more)

Catalogs may use section to subdivide the textual contents of a catalog.

  • {% highlight json %} { "id" : "simple-example", "model-version" : "0.99", "title" : "A Miniature Catalog", "controls" : [ { "id" : "single", "title" : "A Single Control" } ] } {% endhighlight %}

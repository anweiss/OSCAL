Metadata group: meta-group element

Custom- or application-defined annotated metadata.

The meta-group element has the following contents (in order):

  • term attribute (optional)
  • type attribute (optional)
  • meta element (zero or more)
  • meta-group element (zero or more)
  • notes element (zero or one)

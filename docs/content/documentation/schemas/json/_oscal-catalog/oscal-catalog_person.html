Person: person object

A person, with contact information

The person object has the following members (properties):

  • person-name object (zero or one)
  • short-name object (zero or one)
  • org-name object (zero or one): Affiliated organization
  • person-id object (zero or more)
  • org-id object (zero or more)
  • address object (zero or one)
  • email object (zero or more)
  • phone object (zero or more)
  • url object (zero or more)
  • notes object (zero or one)

Organization: org object

An organization or legal entity (not a person), with contact information

The org object has the following members (properties):

  • org-name object (zero or one)
  • short-name object (zero or one)
  • org-id object (zero or more)
  • address object (zero or one)
  • email object (zero or more)
  • phone object (zero or more)
  • url object (zero or more)
  • notes object (zero or one)

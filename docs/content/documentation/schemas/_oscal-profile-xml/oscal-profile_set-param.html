Parameter Setting: set-param element

A parameter setting, to be propagated to points of insertion

The set-param element has the following contents (in order):

  • id attribute (required)
  • class attribute (optional)
  • depends-on attribute (optional)
  • label element (zero or one)
  • desc element (zero or more)
  • constraint element (zero or more)
  • A choice between
    • value element (zero or one)
    • select element (zero or one)
  • link element (zero or more)
  • part element (zero or more)

This element appears inside: modify.

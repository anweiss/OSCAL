Address: address element

A postal address

The address element has the following contents (in order):

  • type attribute (optional)
  • addr-line element (zero or more)
  • city element (zero or one)
  • state element (zero or one)
  • postal-code element (zero or one)
  • country element (zero or one)

This element appears inside: person, org.

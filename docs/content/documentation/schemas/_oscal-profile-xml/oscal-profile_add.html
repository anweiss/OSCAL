Addition: add element

Specifies contents to be added into controls or subcontrols, in resolution

The add element has the following contents (in order):

  • position attribute (optional)
  • title element (zero or one)
  • param element (zero or more)
  • prop element (zero or more)
  • link element (zero or more)
  • part element (zero or more)
  • ref-list element (zero or one)

This element appears inside: alter.

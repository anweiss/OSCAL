Person: person element

A person, with contact information

The person element has the following contents (in order):

  • person-name element (zero or one)
  • short-name element (zero or one)
  • org-name element (zero or one): Affiliated organization
  • person-id element (zero or more)
  • org-id element (zero or more)
  • address element (zero or more)
  • email element (zero or more)
  • phone element (zero or more)
  • url element (zero or more)
  • notes element (zero or one)

This element appears inside: party.

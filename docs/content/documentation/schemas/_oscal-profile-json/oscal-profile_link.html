Link: link string object

This object appears unlabelled in any array called links

Supports Markdown

A line or paragraph with a hypertext link

The link object may have (string) properties

  • href property (optional)
  • rel property (optional)

Works like an HTML anchor (a) except this is a line-oriented (block) element.

This object appears as a property on: ref-list, param, part, set-param, add.
